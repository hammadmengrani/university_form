# **App Name**: AdmitPro

## Core Features:

- Multi-Step Admission Form: A comprehensive 5-step form for applicants to submit their personal, academic, and program information.
- Document Upload: Securely upload required documents to Supabase Storage, with progress tracking and status indicators.
- Application Status Tracking: Allow applicants to track their admission status using their Application ID or CNIC.
- Admin Dashboard: Provide an admin interface for managing applications, staff, and generating reports.
- Review and Decision Workflow: Enable staff to review applications, assign merit scores, and make admission decisions.
- Staff Management: Manage staff users with different roles and permissions.
- Automated Notifications: Automatically send emails to applicants and administrators for application submissions, status changes and general system updates.

## Style Guidelines:

- Primary color: Deep navy (#0A1628) to create a professional and trustworthy feel, reflecting the seriousness of the admission process.
- Background color: Very light, desaturated blue (#E6E9ED) to complement the dark primary and provide a clean, modern backdrop.
- Accent color: Soft gold (#D4B866), a slightly lighter, brighter hue than requested by the user, that provides visual interest to contrast with the primary, indicating important actions or highlighting key information.
- Headline font: 'Playfair', a serif font, providing a contemporary and sophisticated feel that pairs nicely with body content. Note: currently only Google Fonts are supported.
- Body font: 'PT Sans', a sans-serif font, providing a warmer, humanist counterpart for comfortable reading, making the body content more accessible and approachable. Note: currently only Google Fonts are supported.
- Use a consistent set of icons related to education and administration. Use the gold secondary color for icon highlights.
- Employ a clean and organized layout with clear visual hierarchy. Use a grid system to maintain consistency and responsiveness across different screen sizes.
- Use subtle animations and transitions to provide feedback and enhance user experience. For example, use a smooth transition when moving between form steps or a loading animation when submitting data.
